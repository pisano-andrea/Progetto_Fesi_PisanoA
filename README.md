# Progetto_Fesi_PisanoA

Questo progetto Ã¨ caratterizzato da un'idea particolare e ambiziosa:
Dato un segnale musicale in input, studiarlo e poterlo scomporre nelle varie componenti che compongono il segnale (es: partendo dal segnale di una canzone suonata da un'orchestra, poterlo dividere nel segnale composto dai violini, da quello composto dalla batteria, da quello composto dalle chitarre ecc. ecc.).
